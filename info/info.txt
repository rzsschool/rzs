Post Type Slug *
Тип повідомлення Slug *

The post type name/slug. Used for various queries for post type content.
Ім'я/слаг типу публікації. Використовується для різних запитів щодо вмісту типу публікації.

Slugs should only contain alphanumeric, latin characters. Underscores should be used in place of spaces. Set "Custom Rewrite Slug" field to make slug use dashes for URLs.
Слизні мають містити лише буквено-цифрові та латинські символи. Замість пробілів слід використовувати символ підкреслення. Встановіть поле «Custom Rewrite Slug», щоб слиз використовував тире для URL-адрес.
